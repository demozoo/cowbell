Cowbell

a universal web audio player for demoscene music

Cowbell is an HTML / JavaScript audio player focused on demoscene and tracker music. It aims to bring a wide variety of music file formats - from the obvious ones (MP3, OGG) through the classics (MOD, S3M, XM) to the more 'niche' (PT3, VTX...) - together under a common API and user interface.
